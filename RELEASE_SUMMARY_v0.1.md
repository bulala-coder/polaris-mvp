# Polaris v0.1 Release Summary

## Version
Polaris v0.1.7

## Live Demo
https://polaris-mvp-theta.vercel.app

## Repository
https://github.com/bulala-coder/polaris-mvp

## What Polaris v0.1 includes
- A deployed frontend-only prototype
- Today decision dashboard
- Portfolio allocation and exposure dashboard
- Market risk context dashboard
- Journal reflection page
- Settings page with product boundaries and disclaimer
- Vercel SPA routing support

## What Polaris v0.1 does not include
- Real user accounts
- Real portfolio input
- Real market data API
- Database storage
- Brokerage connection
- Financial advice
- Stock recommendations
- Short-term trading signals
- Guaranteed investment outcomes

## v0.1 Product Goal
Polaris v0.1 demonstrates the core product direction:
help long-term investors make calmer, more disciplined, and more explainable decisions.

## Suggested next phase
Polaris v0.2 can focus on editable mock data, local storage, and a more explicit decision engine.
